Этот проект предназначен для обеспечения следующего функционала.
1. Пусть есть блокчейн компания, которая регулярно обменивается токенами со своими клиентами
2. Для поошрения особо отличившихся клиентов компания выпускает специальное NFT с картинкой.
3. Это NFT попадает в кошелек пользователя и красиво в нем отображается.
4. Работа идет в сети TRON и поэтому для кошелелька используется TronLink.
5. В репозитории есть:
 - смарт контракт и скрипты деплоя
 - REST сервис для работы с картинками. В этом сервисе есть модуль, который сохраняет изображение NFT в IPFS
 - есть админка для управления изображениями
